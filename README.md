# KaldrClaw

Claude Code plugin for building and running an AI agency ecosystem. Part of the Kaldr tooling suite.

---

## What it does

KaldrClaw is a pre-configured Claude Code plugin bundle that ships a set of specialized AI agents you can invoke on demand from the Claude Code CLI.

Drop the plugin into Claude Code, invoke an agent by name, get production-quality output.

---

## Stack

- TypeScript
- Claude Code plugin architecture
- MCP (Model Context Protocol) server patterns
- Anthropic Agent SDK
